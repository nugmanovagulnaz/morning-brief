# Morning Brief Bot

Sends a daily Telegram message at 07:00 UTC with London weather, your Google Calendar events, and a health tech news brief generated by Claude.

## Setup

### 1. Clone and install

```bash
git clone <your-repo>
cd morning-brief
npm install
```

### 2. Create a Telegram bot

1. Message [@BotFather](https://t.me/BotFather) on Telegram → `/newbot`
2. Copy the **bot token** → `TELEGRAM_BOT_TOKEN`
3. Start a chat with your bot (or add it to a group), then call:
   ```
   https://api.telegram.org/bot<TOKEN>/getUpdates
   ```
   to find your **chat ID** → `TELEGRAM_CHAT_ID`

### 3. Set up a Google service account

1. Go to [Google Cloud Console](https://console.cloud.google.com/) → **APIs & Services** → **Credentials**
2. Create a **Service Account**, download the JSON key
3. Copy `client_email` → `GOOGLE_CLIENT_EMAIL`
4. Copy `private_key` → `GOOGLE_PRIVATE_KEY` (keep the `\n` characters)
5. Enable the **Google Calendar API** for the project
6. Open Google Calendar → share your calendar with the service account email (Viewer permission)
7. Copy the calendar ID (Settings → Integrate calendar) → `GOOGLE_CALENDAR_ID`

### 4. Get an Anthropic API key

Sign up at [console.anthropic.com](https://console.anthropic.com) → copy your API key → `ANTHROPIC_API_KEY`

### 5. Configure environment variables

Copy `.env.example` to `.env.local` and fill in all values for local development.

### 6. Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Add all environment variables in **Vercel → Project → Settings → Environment Variables**.

Set `CRON_SECRET` to any random string (e.g. `openssl rand -hex 32`). Vercel will automatically attach it as `Authorization: Bearer <CRON_SECRET>` on each cron invocation.

The cron schedule (`0 7 * * *`) is defined in `vercel.json` and is picked up automatically on deploy.

### 7. Test manually

In the Vercel dashboard → **Deployments** → click your deployment → **Functions** tab → find `/api/morning-brief` → **Invoke**. Or trigger it with:

```bash
curl -X GET https://your-project.vercel.app/api/morning-brief \
  -H "Authorization: Bearer your-cron-secret"
```

## Customising the news brief

Edit [`config/prompt.js`](config/prompt.js) — no other files need to change.

## Project structure

```
api/morning-brief.js   Cron handler
lib/weather.js         Open-Meteo weather (London)
lib/calendar.js        Google Calendar events
lib/claude.js          Claude + web search news brief
lib/telegram.js        Telegram message sender
config/prompt.js       News brief prompt (edit me)
vercel.json            Cron schedule
.env.example           Required environment variables
```
